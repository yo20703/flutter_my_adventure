# flutter_my_adventure
#主要是文字冒險遊戲，以**我的桐人**為範本 https://mykirito.com/

我的桐人特性:
* 一直轉生的遊戲，死亡時可以轉生，選擇目前可以使用的角色

* 練級、挑戰敵人時列出list打贏增加經驗值

* 獲得角色的方式
   * 選項會影響獲得甚麼角色

* 對戰方式
   * 完全文字，使用者的選擇只有挑戰誰，對戰的內容系統會自己計算
   
本遊戲走向:
* 及時對戰，攻擊回合
  * 半回合制
  * 血量條
  * 攻擊蓄力條(攻速會影響)
  * 魔量條(回魔能力值會影響)
  * 暫定2個技能(需耗魔)
* 能力值(楓之谷類型)，升等可以點
* 一層一層闖關
